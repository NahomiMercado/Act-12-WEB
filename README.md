# ChuckJokesVue

This project is a simple Vue application created for an academic activity.

## Description
The application displays a list of Chuck Norris jokes using Vue.  
The data is stored in the Vue instance and displayed dynamically with the `v-for` directive.

## Features
- Vue application with a data array
- Iteration with `v-for`
- Styled user interface with CSS
- Responsive card design

## Technologies used
- HTML5
- CSS3
- JavaScript
- Vue.js
